# Simons game 


Simon is an electronic game of short-term memory skill
Play the classic Simon memory game

## Deployment

To deploy this project Download the files 

and then

1.Go To BACKEND folder then

```bash
  npm i
```

then

```bash
    npm start
```

2.Go To FRONTEND folder then

```bash
  npm i
```

then

```bash
    npm run dev
```
